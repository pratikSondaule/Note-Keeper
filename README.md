# Note Keeper
 
It is a simple note keeper app where you can write your notes and other important thing that you want to write.

## Tools used

- HTML
- CSS
- Bootstrap
- JavaScript
- EJS
- Node JS
- Express JS

## What we see in this project?

- Home
- About
- Compose
- Contact

## How to use?

Simply download the repository and run command
### `node app.js`

Or 

### `nodemon app.js`
